# 🔐 Digital Footprint Visualizer

## 📌 Overview
This is a basic web-based project that demonstrates how a user's digital footprint can be visualized using simple charts.

The project uses simulated data to show how online activities (like social media usage, emails, and comments) contribute to a user's digital presence.

## 🎯 Purpose
- Understand the concept of digital footprint
- Visualize user activity using charts
- Demonstrate basic cybersecurity awareness

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js

## ⚙️ Features
- User input (username simulation)
- Simple data visualization using bar chart
- Clean and basic UI

## 🚀 How to Run
1. Download or clone this repository
2. Open `index.html` in any browser
3. Click "Analyze" to see the chart

## 📊 Note
This project uses **simulated data only** and does not collect or process real user information.
